# 多元统计 Multivariate Statistics

> 歧视诉讼中，被告说"我们不是因为性别，是因为经验不同"。你怎么检验这个说法？答案是 **多元回归**——控制其他变量后，看性别是否仍然显著影响薪酬。这是律师在现代诉讼中最常遇到的统计工具，也是对方专家证人最常攻击的靶子。这一章教你 **怎么读懂回归输出、怎么挑战回归、怎么问正确的问题**。

## 1. 为什么律师需要多元统计

- **歧视诉讼**（薪酬、晋升、录用）：隔离受保护特征的纯效应
- **反垄断**：价格效应的实证估计
- **证券诉讼**：事件研究的市场模型（见 `finance.md`）
- **损害赔偿**：基于回归的收入损失预测
- **环境诉讼**：暴露与疾病的关联分析
- **政策评估**：新规则的因果效应
- **政治诉讼**：选区划分、种族投票分析

## 2. 从散点图开始

### 2.1 散点图 Scatterplots

**探索两个变量关系的第一步**。

绘制方法：
1. X 轴 = 解释变量 (independent / explanatory variable)
2. Y 轴 = 响应变量 (dependent / response variable)
3. 每个观测一个点

**揭示**：线性、非线性、无关系、异常值、异方差。

**律师习惯**：**先画图再回归**。很多错误的回归能被一张散点图预警。

### 2.2 Anscombe's Quartet

著名的教学例子：**四个数据集** 有完全相同的均值、方差、相关系数、回归直线——但散点图截然不同。一个是线性关系、一个是抛物线、一个是有一个极端异常点的线性关系、一个是 X 的一个值加几个点。

**教训**：**数字统计量不能代替可视化**。对方专家只给你数字不给你图？那就自己画。

## 3. 相关 Correlation

### 3.1 Pearson 相关系数 $r$

衡量 **线性关系** 的方向和强度：

$$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}$$

- $r \in [-1, 1]$
- $r > 0$：正相关；$r < 0$：负相关；$r = 0$：无线性相关
- $|r|$ 越大，线性关系越强

| $\|r\|$ | 强度 |
|---|---|
| $< 0.1$ | 几乎无关 |
| $0.1$–$0.3$ | 弱 |
| $0.3$–$0.5$ | 中 |
| $\geq 0.5$ | 强 |

### 3.2 其他相关系数

- **Spearman 秩相关 $\rho$**：基于排序，**不需要线性假设**。对单调但非线性的关系更稳健。
- **Kendall's $\tau$**：另一种秩相关
- **Point-biserial**：连续变量 vs 二元变量

### 3.3 关键警告

- **$r$ 只捕捉线性关系**。$r = 0$ 不等于"无关"——可能是非线性关系（U 形、倒 U 形）
- **对异常值极敏感**：一个离群点可以把 $r$ 从 $0$ 拉到 $0.5$
- **相关不是因果**（§4 详述）

## 4. 相关 ≠ 因果

**这是统计学中最重要、律师最需要牢记的原则**。两个变量一起变动，不代表一个导致了另一个。

### 4.1 相关但无因果的四种情况

1. **巧合 (spurious correlation)**：纯巧合——足够大的数据集里总能找到
2. **反向因果 (reverse causation)**：Y 导致 X 而不是 X 导致 Y
3. **共同原因 (common cause / confounding)**：Z 同时导致 X 和 Y
4. **选择效应 (selection effect)**：样本被系统性地选择

### 4.2 经典例子

**冰淇淋销量与溺水事故**：正相关——**共同原因** 是气温。不是冰淇淋导致溺水。

**警察数量与犯罪率**：正相关——因为高犯罪率的城市派驻更多警察。**反向因果**。不控制这一点会得出"警察增加犯罪"的荒谬结论。

**就业与教育**：高学历者失业率低——但可能因为 **能力（未观察）** 同时导致高学历和低失业。**混杂变量**。

**酗酒与离婚**：都相关——但可能都受 **婚姻不幸福、经济压力** 影响。

### 4.3 因果推断的黄金标准：随机对照试验 (RCT)

随机分配处理 → 消除 **选择偏差** 和 **混杂变量**。但在法律问题中，RCT 往往不可行（伦理、实际、成本）。所以需要 **准实验方法**（§13）。

## 5. 简单线性回归 Simple Linear Regression

### 5.1 模型

$$Y = \alpha + \beta X + \epsilon$$

- $\alpha$ = 截距 (intercept)：当 $X = 0$ 时 $Y$ 的期望值
- $\beta$ = 斜率 (slope)：$X$ 每增加一单位，$Y$ 的期望变化
- $\epsilon$ = 误差项：无法被 $X$ 解释的部分

### 5.2 最小二乘估计 OLS

选择 $\hat{\alpha}, \hat{\beta}$ 使 **残差平方和** 最小：

$$\min \sum_{i=1}^{n} (y_i - \hat{\alpha} - \hat{\beta} x_i)^2$$

闭式解：

$$\hat{\beta} = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = r \cdot \frac{s_y}{s_x}$$

$$\hat{\alpha} = \bar{y} - \hat{\beta} \bar{x}$$

### 5.3 Gauss-Markov 定理

**OLS 是 BLUE (Best Linear Unbiased Estimator)**——在几个假设下方差最小：
1. **线性关系** (linearity)
2. **误差均值为零** ($E[\epsilon] = 0$)
3. **同方差** (homoscedasticity)：误差方差恒定
4. **无自相关** (no autocorrelation)：$\text{Cov}(\epsilon_i, \epsilon_j) = 0$
5. **$X$ 是非随机的 / 与 $\epsilon$ 不相关**
6. **（可选）正态性**：假设检验时需要

**假设失败会怎样**？见 §10。

## 6. $R^2$ 决定系数

$$R^2 = 1 - \frac{\text{SSR}}{\text{TSS}} = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$$

- **TSS** = Total Sum of Squares = $Y$ 的总变异
- **SSR / RSS** = Residual Sum of Squares = 模型无法解释的变异
- **ESS** = Explained Sum of Squares = 模型解释的变异

$R^2$ 的解读：**模型解释了 $Y$ 变异的多大比例**。

- $R^2 = 0.80$ → $80\%$ 的 $Y$ 变异被模型解释，$20\%$ 是噪声或遗漏变量
- $R^2 = 0.20$ → 模型只解释 $20\%$

**经济 / 社会数据**：$R^2 = 0.3$–$0.6$ 常见
**物理学数据**：$R^2 > 0.9$ 常见

**关键警告**：**高 $R^2$ 不等于好模型**。你可以通过加入无关变量人为提高 $R^2$。**低 $R^2$ 也不等于回归系数不显著**——关心的是系数本身。

### 6.1 调整 $R^2$

惩罚加入更多变量：

$$\bar{R}^2 = 1 - (1 - R^2) \cdot \frac{n - 1}{n - k - 1}$$

（$k$ = 解释变量个数）

**用法**：比较不同变量数的模型时用 $\bar{R}^2$，不要用原始 $R^2$。

## 7. 多元线性回归 Multiple Linear Regression

### 7.1 模型

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_k X_k + \epsilon$$

**核心优势**：**控制其他变量后** 隔离某个变量的独立效应。

### 7.2 系数的解读

$\beta_j$ = **在保持其他所有变量不变的情况下**，$X_j$ 每增加一单位，$Y$ 的期望变化。

**这就是回归在歧视诉讼中的作用**：控制学历、工龄、职位后，**性别** 还有没有显著效应？

### 7.3 Worked Example：薪酬歧视（简化版）

**模型**：

$$\text{Salary}_i = \beta_0 + \beta_1 \text{Gender}_i + \beta_2 \text{Education}_i + \beta_3 \text{Experience}_i + \beta_4 \text{Department}_i + \epsilon_i$$

（Gender: 男 = 1, 女 = 0；虚拟编码）

**假设结果**：
- $\hat{\beta}_1 = 5{,}200$，标准误 $= 1{,}100$
- $t = 5{,}200 / 1{,}100 = 4.73$
- $p < 0.001$

**解读**：**控制学历、经验、部门后，男性平均年薪比女性高 $\$5{,}200$**，高度显著。

**原告 (plaintiff)**：这是歧视的 prima facie 证据。
**被告 (defendant)**：反驳方向——**遗漏变量** (§10.1)，比如"工作兴趣"、"可用时间"、"特定技能"。

## 8. 虚拟变量 Dummy Variables

### 8.1 基本用法

**分类变量** 不能直接进回归。用 **虚拟变量 (dummy / indicator variable)** 编码：

| 分类 | $D_1$ | $D_2$ | $D_3$ |
|---|---|---|---|
| 男性 | 1 | 0 | 0 |
| 女性 | 0 | 1 | 0 |
| 其他 | 0 | 0 | 1 |

**陷阱**：**虚拟变量陷阱 (dummy variable trap)**。如果 $k$ 个类别都加进去，它们加起来恒等于 1，与截距项 **完全共线**——回归矩阵不满秩，无解。

**解法**：去掉一个类别作为 **基准 (reference group)**，剩下 $k - 1$ 个虚拟变量。系数解读为"与基准的差距"。

### 8.2 交互项 Interaction Terms

**基本交互项**：

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 (X_1 \cdot X_2) + \epsilon$$

$\beta_3 > 0$ 表示 $X_1$ 的效应 **随 $X_2$ 增加而增加**。

**在歧视诉讼中**：**性别 $\times$ 经验** 可以检验"是否经验对女性的回报更低"——这是比单独性别系数更微妙的歧视形式。

## 9. 假设检验 in Regression

### 9.1 t 检验（单个系数）

$$t = \frac{\hat{\beta}_j - 0}{SE(\hat{\beta}_j)}$$

自由度 $df = n - k - 1$。

**显著性判断**：
- $|t| > 1.96$（近似）→ $p < 0.05$（双尾）
- $|t| > 2.58$ → $p < 0.01$
- $|t| > 3.29$ → $p < 0.001$

### 9.2 F 检验（联合显著性）

检验 **一组系数同时为零** 的假设：

$$H_0: \beta_1 = \beta_2 = \cdots = \beta_k = 0$$

$$F = \frac{R^2 / k}{(1 - R^2) / (n - k - 1)}$$

**应用**：测试整个模型是否有解释力；测试一组虚拟变量（比如所有部门）是否联合显著。

## 10. OLS 假设失败及后果

### 10.1 遗漏变量偏误 Omitted Variable Bias

**这是歧视诉讼中最常见、最致命的问题**。

如果真实模型是：

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \epsilon$$

但你只跑：

$$Y = \gamma_0 + \gamma_1 X_1 + u$$

则 $\hat{\gamma}_1$ 的期望是：

$$E[\hat{\gamma}_1] = \beta_1 + \beta_2 \cdot \frac{\text{Cov}(X_1, X_2)}{\text{Var}(X_1)}$$

**偏误方向**：$\beta_2$ 与 $X_1, X_2$ 的相关性决定偏误方向。

**例 (Sears 案)**：Sears 的原告模型没包含"对该工作的兴趣"。如果男性对销售佣金工作兴趣更大，而兴趣也影响薪酬，那么 **性别系数** 就被偏误——把"兴趣效应"归给了"性别"。

**律师策略**：
- **原告**：论证所有"合理"的控制变量都已包含；遗漏的变量不太可能与性别相关
- **被告**：提出未控制的"合法"变量；说明它们与性别相关

### 10.2 多重共线性 Multicollinearity

**定义**：两个或多个解释变量 **高度相关**。

**症状**：
- 单个系数的 $t$ 不显著，但 F 检验显著
- 系数的符号与预期相反
- 系数对加 / 减一个变量极敏感
- 标准误非常大

**诊断**：
- **相关矩阵**：变量间相关系数
- **VIF (Variance Inflation Factor)**：$\text{VIF}_j = 1 / (1 - R_j^2)$，$R_j^2$ 是 $X_j$ 对其他解释变量回归的 $R^2$
- **VIF > 10** 通常被视为严重问题

**例**：同时包含 **年龄** 和 **工龄**——高度相关，难以分离独立效应。

**解法**：
- 去掉一个
- 合并为一个（如平均值）
- 主成分分析 (PCA)
- 用更大样本（有时能减轻，但不总是）
- **接受宽置信区间**（有时这就是真相）

### 10.3 异方差 Heteroscedasticity

**定义**：误差的方差 **不恒定**（$\text{Var}(\epsilon_i)$ 随 $X_i$ 变化）。

**例**：薪酬的方差随经验增加而增加（高级岗位的薪酬差距大）。

**后果**：
- OLS 系数仍然 **无偏**
- 但 **标准误不正确**——导致假设检验 **无效**（可能误判显著性）

**诊断**：
- **Breusch-Pagan 检验**
- **White 检验**
- **残差对 $\hat{Y}$ 的散点图**：扇形、漏斗形是症状

**解法**：
- **稳健标准误 (robust / Huber-White SE)**：现代标准做法
- **加权最小二乘 (WLS)**
- **变换变量**（取对数常常有效）

### 10.4 自相关 Autocorrelation

**定义**：误差之间相关（$\text{Cov}(\epsilon_i, \epsilon_j) \neq 0$）。

**常见场景**：时间序列数据（今天的误差与昨天相关）、聚类数据（同一公司内员工误差相关）。

**诊断**：
- **Durbin-Watson 检验**（时间序列）
- **残差自相关图**

**解法**：
- **Cochrane-Orcutt 变换**
- **Newey-West 标准误**
- **聚类稳健标准误**（cluster-robust SE）

### 10.5 双向因果 / 内生性 Endogeneity

**$X$ 与 $\epsilon$ 相关** → OLS 系数 **有偏**。

**来源**：
1. **反向因果**（$Y$ 影响 $X$）
2. **遗漏变量** 同时影响 $X$ 和 $Y$
3. **测量误差** 在 $X$ 中

**警察与犯罪** 是经典例子。简单回归：

$$\text{犯罪率} = \alpha + \beta \cdot \text{警察数} + \epsilon$$

会发现 $\hat{\beta} > 0$ —— 但这不代表 **警察增加犯罪**，而是因为高犯罪率的地方 **部署** 更多警察。

**解法**：工具变量 (§13.1)

### 10.6 异常值 Outliers & Influential Points

- **异常值 (outlier)**：残差特别大
- **高杠杆点 (high leverage)**：$X$ 值特别极端
- **影响点 (influential point)**：同时异常 + 高杠杆，拉动回归直线

**诊断**：
- **标准化残差**：> $|3|$ 是异常
- **Cook's distance**：> $4/n$ 值得检查
- **Leverage statistic (hat value)**

**处理**：
- 调查原因（数据录入错误？真实但罕见？）
- 报告 **有和没有** 异常值的结果
- 不能仅因"不好看"就剔除

## 11. 非线性与变量变换

### 11.1 对数变换 Log Transformation

**对数-线性模型 (log-linear)**：

$$\ln Y = \beta_0 + \beta_1 X + \epsilon$$

系数解读：**$X$ 每增加一单位，$Y$ 增加 $100 \beta_1\%$**（近似）。

**对数-对数模型 (log-log / elasticity)**：

$$\ln Y = \beta_0 + \beta_1 \ln X + \epsilon$$

系数解读：**弹性**——$X$ 变化 $1\%$，$Y$ 变化 $\beta_1\%$。

**律师应用**：
- 薪酬回归常用 $\log(\text{salary})$——因为薪酬分布右偏
- 价格弹性估计用 log-log
- 企业规模效应用 log(revenue)

### 11.2 多项式项 Polynomial Terms

$$Y = \beta_0 + \beta_1 X + \beta_2 X^2 + \epsilon$$

捕捉 **U 形** 或 **倒 U 形** 关系。例：经验与薪酬——起初快速增长，然后趋于平缓甚至下降（退休前）。

### 11.3 分段线性 / 样条 Piecewise / Splines

在不同区间用不同斜率。适合阈值效应。

## 12. Logistic 回归 Logistic Regression

### 12.1 为什么需要

OLS 不适合 **二元结果**（$Y = 0$ 或 $1$）——预测值可能 < 0 或 > 1，误差不是正态。

### 12.2 模型

$$P(Y = 1 \mid X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \cdots)}}$$

等价于：

$$\ln \left( \frac{P}{1 - P} \right) = \beta_0 + \beta_1 X_1 + \cdots$$

**左边** 叫 **对数几率 (log odds / logit)**。

### 12.3 Odds Ratio 的解读

$$\text{OR} = e^{\beta_j}$$

**OR** = 2 表示 $X_j$ 增加一单位，$Y = 1$ 的 **几率（odds）** 翻倍——**不是概率翻倍**。

### 12.4 法律应用

- **录用 / 晋升**：$Y$ = 是否录用
- **刑事量刑**：$Y$ = 是否判监禁
- **破产预测**：$Y$ = 是否破产
- **疾病诊断**：$Y$ = 是否患病

## 13. 因果推断工具箱 Causal Inference Toolkit

当无法做 RCT 时，现代准实验方法：

### 13.1 工具变量 Instrumental Variables (IV)

**问题**：$X$ 与 $\epsilon$ 相关（内生），OLS 有偏。

**解法**：找一个 **工具 $Z$**，满足：
1. **相关性 (relevance)**：$Z$ 与 $X$ 相关
2. **外生性 (exogeneity)**：$Z$ 与 $\epsilon$ 无关（$Z$ 只通过 $X$ 影响 $Y$）

**两阶段最小二乘 (2SLS)**：
- 第一阶段：$X = \pi_0 + \pi_1 Z + v$ → 得 $\hat{X}$
- 第二阶段：$Y = \beta_0 + \beta_1 \hat{X} + u$

**经典例子**：
- **Angrist-Krueger (1991)**：出生季节作为受教育年数的工具（影响入学年龄 → 影响受教育年数，但对成年收入无直接影响）
- **Levitt (1996)**：监狱人口上限诉讼作为监狱人口的工具，估计监禁对犯罪的因果效应
- **Acemoglu-Johnson-Robinson (2001)**：殖民者死亡率作为制度质量的工具

**律师应用**：当对方的回归存在明显内生性时，问 **"为什么不用 IV？找不到合适的工具吗？"**

### 13.2 Difference-in-Differences (DiD)

**场景**：某项政策在 $t$ 时刻实施于某组（处理组），另一组未受影响（对照组）。

**DiD 估计量**：

$$\text{DiD} = (Y_{\text{post, treated}} - Y_{\text{pre, treated}}) - (Y_{\text{post, control}} - Y_{\text{pre, control}})$$

**核心假设**：**平行趋势 (parallel trends)**——如果没有政策，两组会以相同速度变化。

**例**：
- **Card-Krueger (1994)**：新泽西最低工资上调对就业的影响（对照组：宾夕法尼亚）——颠覆了"最低工资必然减少就业"的传统观点
- **Ashenfelter (1978)**：培训项目的效果

**律师应用**：评估新规则、新判例的经济效果；损害赔偿中的 but-for 分析。

**陷阱**：平行趋势假设 **不可直接检验**（只能事前看图形）。

### 13.3 断点回归 Regression Discontinuity (RD)

**场景**：处理与否取决于某个阈值（如学生考分 > 70 进入资优班）。

**逻辑**：阈值附近的两组基本相同，**唯一区别** 是处理与否——类似局部 RCT。

**例**：
- **最低投票年龄、饮酒年龄**：年龄 18 岁 ± 1 个月的人"几乎相同"，但法律地位完全不同
- **选举政治**：险胜 vs 险败的候选人

### 13.4 倾向得分匹配 Propensity Score Matching (PSM)

**基本想法**：为每个处理组成员找一个 **尽可能相似** 的对照组成员。

**步骤**：
1. 用 logistic 回归估计 **倾向得分** $P(\text{Treated} = 1 \mid X)$
2. 匹配得分相近的个体
3. 比较匹配对的 $Y$

**用途**：处理 **选择偏差**——但不能处理 **未观察的混淆变量**（IV 可以）。

### 13.5 合成控制法 Synthetic Control

**场景**：只有一个处理单元（如一个城市、一个州、一个国家）。

**方法**：用其他单元的加权组合构造一个"合成对照"，使其在处理前与真实处理单元相似。

**例**：**Abadie-Gardeazabal (2003)**：巴斯克地区恐怖主义对 GDP 的影响。

## 14. 歧视诉讼中的回归：完整对抗清单

### 14.1 原告的经典步骤

1. 建立 **基准回归**：$Y$ = 薪酬（或 log 薪酬）
2. **关键变量**：受保护特征（性别、种族、年龄）
3. **控制变量**：所有 **合法** 的影响因素（教育、经验、部门、职级）
4. 报告系数、标准误、$t$、$p$、$R^2$
5. **稳健性检验**：不同样本、不同控制集、不同函数形式
6. **交互项**：检验歧视的异质性

### 14.2 被告的经典反驳

1. **遗漏变量**：缺少关键合法变量（工作时数、特定技能、兴趣、客观绩效）
2. **样本选择**：样本不是随机的
3. **多重共线性**：系数估计不稳定
4. **双向因果**：薪酬 → 流动 → 观察到的"经验"（薪酬高的留下来的经验更多）
5. **功能形式**：线性假设错误
6. **异常值驱动**：少数高薪或低薪观察驱动结果
7. **分层效应 / Simpson 悖论**：各部门内部性别差距都不显著，只是部门间分布导致的
8. **比较错误**：应与同职级 / 同部门比较，而非总体

### 14.3 回归质量检查表

1. **样本**：代表性？样本量足够吗？
2. **变量选择**：所有合理控制变量都在吗？
3. **函数形式**：线性合适？对数？多项式？
4. **虚拟变量**：正确编码？基准组合理？
5. **交互项**：检查了异质性吗？
6. **多重共线性**：VIF 检查
7. **异方差**：用了稳健 SE 吗？
8. **异常值**：诊断并处理
9. **遗漏变量**：有没有理论上重要但未包含的变量？
10. **内生性**：有没有反向因果？需要 IV 吗？
11. **模型稳健性**：换规范后结果稳定吗？
12. **预测 vs 因果**：这个回归是做预测还是论证因果？

## 15. Bazemore v. Friday, 478 U.S. 385 (1986)

**最高法院的关键判例**：回归分析在歧视诉讼中 **必须被接受** 作为证据，即使它不包含 **所有** 可能相关的变量。

**Powell 法官**：*"A plaintiff in a Title VII suit need not prove discrimination with scientific certainty; rather, his or her burden is to prove discrimination by a preponderance of the evidence."*

**含义**：
- 被告不能通过"你没控制所有变量"简单击败原告的回归
- 但被告可以通过显示 **具体的、合理的** 遗漏变量扭曲了结果来反驳
- 法官作为 gatekeeper（Daubert 后）决定哪些遗漏变量的挑战是合理的

## 16. 回归 vs 描述：永远分清楚

一个常见混淆：**回归是在做因果推断还是纯预测**？

- **描述**：这些变量之间如何关联
- **预测**：给定 $X$，最好的 $Y$ 预测是什么
- **因果**：如果我们改变 $X$，$Y$ 会如何变化

**OLS 做得好"描述"和"预测"**（在假设满足时），但 **因果推断** 需要额外的识别假设（随机化、自然实验、IV、DiD 等）。

**律师最常见的错误**：把一个描述性回归当成因果证据。**警察数量与犯罪率的回归** 本身只是描述性的；要说"增加警察能减少犯罪"需要因果识别策略。

## 17. 书中与学术经典案例

- **Bazemore v. Friday (1986)**：回归证据的确立
- **McCleskey v. Kemp (1987)**：Baldus 研究——死刑判决中的种族差异，大型回归分析
- **Price Waterhouse v. Hopkins (1989)**：性别成见 + 回归证据
- **Wal-Mart v. Dukes (2011)**：集体诉讼中大样本回归的挑战
- **Sears 案**：历史上"兴趣"作为遗漏变量的经典辩护
- **Shapiro et al., "Regression Analysis in Title VII Cases" (多篇学术论文)**

## 18. 律师的回归对抗清单（速查）

### 当对方给你一个回归

1. **样本**：怎么选的？代表性？
2. **变量选择**：合理控制变量都在吗？**关键：遗漏变量**
3. **系数的显著性**：$t > 2$ 吗？
4. **效应量**：系数大到有实际意义吗？
5. **$R^2$**：模型解释力如何？
6. **稳健性**：不同规范下结果稳定吗？
7. **OLS 假设**：异方差、内生性、共线性？
8. **因果 vs 描述**：这是哪一种？

### 当你需要反驳

9. 提出具体的、有理有据的 **遗漏变量**
10. 指出 **样本选择** 问题
11. 质疑 **因果识别策略**
12. 自己跑一个替代回归，展示结果不稳定
13. 做 **Sensitivity analysis**：系数要变多少才会不显著？

## 19. 常见陷阱速查表

| 陷阱 | 症状 | 修正 |
|---|---|---|
| 相关当因果 | "X 与 Y 相关，所以 X 导致 Y" | 识别策略、因果工具 |
| 遗漏变量 | 系数被偏误 | 加入遗漏变量或用 IV |
| 双向因果 | 警察-犯罪 | IV、DiD、自然实验 |
| 多重共线性 | $t$ 不稳定 | VIF、合并变量 |
| 异方差 | 标准误错误 | 稳健 SE |
| 虚拟变量陷阱 | 矩阵不满秩 | 去掉一个基准组 |
| 异常值驱动 | 结果不稳健 | Cook's D、稳健回归 |
| 高 $R^2$ 自满 | "模型很好" | 检查系数、假设 |
| 线性假设错误 | 残差图有模式 | 对数、多项式、样条 |
| 样本选择 | 只看某类观察 | Heckman 校正 |
| Simpson 悖论 | 总体 vs 分层相反 | 分层回归 |
| 统计显著 ≠ 实际意义 | $p < 0.05$ 就结论 | 效应量 |
| 描述当因果 | "回归说明 X 导致 Y" | 明确识别策略 |
