# 微观经济学 Microeconomics

> 微观经济学给律师提供了一套理解 **市场如何运作** 以及 **如何失灵** 的框架。反垄断、消费者保护、环境法、税法、监管设计——几乎所有领域都能从供需分析中获益。这一章不是教科书式入门，而是 **律师实战中最常遇到的微观经济工具**。

## 1. 市场基础

### 1.1 竞争市场 Perfectly Competitive Market

**理想化特征**：
- **众多小卖家 & 小买家**：每个人都是"价格接受者"（price taker），没有定价权
- **同质产品**：买家不区分卖家
- **完全信息**
- **自由进出**
- **无外部性**

**现实中几乎不存在** 完美竞争市场，但它是分析偏离的基准。

### 1.2 需求曲线 Demand Curve

**向下倾斜**：价格 $\uparrow$ → 需求量 $\downarrow$。

**移动需求曲线的因素**（与沿曲线移动不同）：
- **收入变化**：正常品（收入 $\uparrow$ → 需求 $\uparrow$）、劣等品（相反）
- **替代品 (substitutes) 价格**：牛肉涨价 → 猪肉需求 $\uparrow$
- **互补品 (complements) 价格**：汽油涨价 → 汽车需求 $\downarrow$
- **偏好**、**人口**、**预期**

### 1.3 供给曲线 Supply Curve

**向上倾斜**：价格 $\uparrow$ → 愿意供给更多。

**短期 vs 长期**：短期内供给曲线更 **陡峭**（产能受限），长期可以调整资本。

**移动供给曲线的因素**：投入成本、技术、生产者数量、自然条件、法规。

### 1.4 均衡 Equilibrium

供给和需求曲线的 **交点** = 市场出清价格 $P^*$ 和数量 $Q^*$。

## 2. 弹性 Elasticity

### 2.1 价格弹性 Price Elasticity of Demand

衡量需求量对价格变化的敏感度：

$$\epsilon_d = \left| \frac{\% \Delta Q^d}{\% \Delta P} \right|$$

| 弹性 | 称呼 | 涨价对 **总收入** 的影响 |
|---|---|---|
| $\epsilon > 1$ | 有弹性 (elastic) | 收入 $\downarrow$（量降得多） |
| $\epsilon < 1$ | 无弹性 (inelastic) | 收入 $\uparrow$（量降得少） |
| $\epsilon = 1$ | 单位弹性 | 收入不变 |

**影响弹性的因素**：
- **替代品可得性**：替代多 → 弹性大
- **必需品 vs 奢侈品**：必需品弹性小
- **预算占比**：占比小 → 弹性小（salt vs 汽车）
- **时间跨度**：长期弹性 > 短期弹性

### 2.2 其他弹性

- **收入弹性**：$\epsilon_I = \% \Delta Q / \% \Delta I$（正 = 正常品，负 = 劣等品）
- **交叉价格弹性**：$\epsilon_{XY} = \% \Delta Q_X / \% \Delta P_Y$（正 = 替代品，负 = 互补品）
- **供给弹性**：$\epsilon_s = \% \Delta Q^s / \% \Delta P$

### 2.3 律师应用

- **反垄断 / 市场界定**：交叉价格弹性决定商品是否在同一个相关市场
- **税负归宿 (tax incidence)**：弹性小的一方承担更多税负——谁的弹性小，谁吃亏
- **损害赔偿**：如果需求无弹性，涨价损害较小；弹性大，损害更大

## 3. 剩余与效率 Surplus & Efficiency

### 3.1 消费者剩余 Consumer Surplus

$$CS = \text{消费者愿支付} - \text{实际支付}$$

在需求-价格图上，是 **需求曲线以下、价格线以上** 的面积。

### 3.2 生产者剩余 Producer Surplus

$$PS = \text{实际收入} - \text{生产成本}$$

是 **价格线以下、供给曲线以上** 的面积。

### 3.3 总剩余 Total Surplus

$$TS = CS + PS = v - c$$

（$v$ = 买方价值，$c$ = 卖方成本）

### 3.4 效率定理

**在竞争均衡数量 $Q^*$ 处，总剩余最大化**。这就是 Adam Smith 的"看不见的手"的数学表达。

任何使实际数量偏离 $Q^*$ 的干预（最低价、最高价、税收、垄断）都会造成 **无谓损失 (deadweight loss, DWL)**——总剩余的净减少。

**律师用途**：反垄断诉讼中的损害赔偿 = 消费者剩余的净减少。

## 4. 政府干预的经济分析

### 4.1 价格下限 Price Floor

设定的最低合法价格，**高于均衡价格** 时生效。

**结果**：供给 > 需求 → **过剩**。

**例**：最低工资（劳动力过剩 = 失业）、农产品保护价（政府收购）。

**DWL**：等于没交易的互利交易的消失剩余。

### 4.2 价格上限 Price Ceiling

设定的最高合法价格，**低于均衡价格** 时生效。

**结果**：需求 > 供给 → **短缺**。

**例**：纽约租金管制、二战物资配给。非价格分配机制（排队、走后门、黑市）出现。

### 4.3 税收

**商品税的税负归宿**：不取决于 **名义上** 谁交税，取决于 **弹性**。

$$\text{买方承担比例} = \frac{\epsilon_s}{\epsilon_s + \epsilon_d}$$

$$\text{卖方承担比例} = \frac{\epsilon_d}{\epsilon_s + \epsilon_d}$$

**直觉**：弹性小的一方"跑不掉"，承担更多税。

**DWL**：税收总是造成 DWL（因为改变了交易量）。DWL 与税率的 **平方** 成正比——小税小问题，大税大问题。

### 4.4 补贴

相当于负税。同样产生 DWL（因为引诱了本不应发生的交易），但可以用来纠正正外部性（教育、疫苗、清洁能源）。

## 5. 垄断 Monopoly

### 5.1 垄断者的定价

垄断者面对的是 **整个市场的需求曲线**（向下倾斜），所以：
- 涨价 → 销量下降
- **边际收益 (MR)** < 价格

**最优化条件**：$MR = MC$（边际收益 = 边际成本）。

对线性需求 $P = a - bQ$，$MR = a - 2bQ$（斜率是需求曲线的两倍）。

### 5.2 垄断的无谓损失

在 $MR = MC$ 的产量上，垄断者收取 **远高于 MC** 的价格。相比竞争均衡：
- 产量 $\downarrow$
- 价格 $\uparrow$
- **消费者剩余** → 部分转移给垄断者（作为垄断利润），部分变成 **DWL**
- **总剩余** $\downarrow$

这就是反垄断法的 **核心经济学论据**。

### 5.3 Lerner 指数

衡量垄断力量：

$$L = \frac{P - MC}{P} = \frac{1}{\epsilon_d}$$

完美竞争 $L = 0$（$P = MC$），纯垄断 $L$ 接近 1。

**律师应用**：如果被告说"我没有市场力量"，你可以从 **毛利率** 反推——毛利率高就意味着 $P$ 远超 $MC$。

## 6. 反垄断的市场界定 Market Definition

反垄断案件的 **第一步永远是定义相关市场**。定义得宽，被告的市场份额就小、看起来没有垄断；定义得窄，份额就大、看起来垄断。

### 6.1 SSNIP Test (Hypothetical Monopolist Test)

**Small but Significant and Non-transitory Increase in Price**。美国和欧盟反垄断局的标准方法。

**逻辑**：从一个候选市场定义开始，假设有一个垄断者能 **统一** 这些产品。如果这个假想垄断者能 **有利可图地涨价 $5$–$10\%$**（通常 $5\%$）一年以上，那这个定义就是"最小的相关市场"。如果不行（因为消费者会转向市场之外的替代品），就扩大市场定义，继续测试。

**实施**：
- **定性分析**：替代品、消费者转换成本、产品特征
- **定量分析**：交叉价格弹性、critical loss analysis
- **证据**：公司内部文件、pricing data、营销文档

### 6.2 Critical Loss Analysis

"涨价 $5\%$ 后需要多少销量损失才会让涨价不划算"：

$$\text{Critical Loss} = \frac{\Delta P / P}{\Delta P / P + \text{毛利率}}$$

如果实际预期损失 > critical loss → 涨价不划算 → 扩大市场。如果 < critical loss → 涨价有利可图 → 市场定义成立。

### 6.3 HHI 赫芬达尔-赫希曼指数

**合并控制 (merger control)** 的关键指标：

$$HHI = \sum_{i=1}^{n} s_i^2$$

（$s_i$ 是第 $i$ 家公司的市场份额百分比）

**阈值（2023 DOJ/FTC Merger Guidelines）**：
- $HHI < 1{,}000$：未集中
- $1{,}000 \leq HHI < 1{,}800$：适度集中
- $HHI \geq 1{,}800$：高度集中
- $\Delta HHI \geq 100$（集中市场中）：可能引发审查

**例**：5 家公司市场份额 $40, 30, 15, 10, 5$：

$$HHI = 40^2 + 30^2 + 15^2 + 10^2 + 5^2 = 1600 + 900 + 225 + 100 + 25 = 2{,}850$$

属于高度集中。

## 7. 价格歧视 Price Discrimination

垄断者向不同消费者收取不同价格，试图榨取更多消费者剩余。

### 7.1 三种程度

**一级价格歧视 (perfect / first-degree)**：对每个消费者收取他的 **最高支付意愿**。完全榨取消费者剩余。现实中罕见——需要完美信息。

**二级价格歧视 (second-degree)**：基于 **数量 / 版本**。同一产品，不同"版本"定不同价。
- 飞机头等舱 vs 经济舱
- 软件基础版 vs 专业版
- 批发 vs 零售

**三级价格歧视 (third-degree)**：基于 **消费者类型 / 市场**。
- 学生票、老年人票
- 国内市场 vs 国际市场（制药公司经典）
- 早鸟优惠

### 7.2 法律问题

- **Robinson-Patman Act** (1936)：禁止某些形式的价格歧视——但执法已经非常弱
- **反垄断滥用**：价格歧视可能是垄断力量的证据
- **消费者保护**：个性化定价（基于算法）引发的公平性争议
- **国际贸易**：倾销 (dumping) 本质是国际价格歧视

### 7.3 Welfare 分析

价格歧视是否降低福利 **取决于**：
- 是否扩大了总产量（好）
- 还是只是把 CS 转给了生产者（中性到坏）

许多学者认为价格歧视在很多情况下 **提高** 总福利——但这在立法辩论中并不总被接受。

## 8. 反竞争行为 Anticompetitive Practices

### 8.1 掠夺性定价 Predatory Pricing

**定义**：降价到成本以下，逼走竞争者，之后抬价收割。

**经济学评价**：**很难成立** 作为理性策略。原因：
- 降价期间的亏损必须被事后的垄断租金补偿回来
- 但一旦抬价，新进入者会再来
- Matsushita Elec. v. Zenith Radio (1986)：最高法院表示掠夺性定价"内在地不可能"

**法律标准 (Brooke Group, 1993)**：
1. 价格 **低于适当的成本度量**（平均可变成本或 AVC 附近）
2. 被告有 **补偿垄断租金的合理可能性**

### 8.2 掠夺性定价的变种

- **排除性交易 (exclusive dealing)**：绑定分销商
- **捆绑 (bundling)**：搭售多个产品
- **搭售 (tying)**：买 A 必须买 B（如 Microsoft 把 IE 和 Windows 绑定）
- **最惠国条款 (MFN)**：承诺"永远给你最低价"——可以抑制其他买家的降价要求

### 8.3 共谋 Collusion

**明示共谋 (explicit collusion)**：卡特尔——美国和欧盟都是 per se 违法，常刑事。

**默契共谋 (tacit collusion)**：没有协议，但通过 **重复博弈** 中的相互威慑维持高价。更难起诉，因为没有"协议"证据。

**Cournot 模型**：产量博弈的均衡比完美竞争价格高，比纯垄断价格低——寡头的典型情境。

**Bertrand 模型**：两家企业的价格竞争会把价格压到 $MC$（Bertrand paradox）——说明了为什么寡头市场往往有隐形的协调机制。

## 9. 外部性 Externalities

### 9.1 定义

一方的行动影响第三方福利，但 **没有通过市场价格反映**。

- **负外部性**：污染、噪音、拥堵
- **正外部性**：教育、疫苗、R&D 的溢出效应

### 9.2 Coase 定理 (1960)

**"在没有交易成本的情况下，无论产权如何初始分配，私人协商都会达到有效率的结果"**。

**直觉**：工厂污染农民的水源。如果农民有权要求清洁水（Property Rule 1），他可以卖给工厂"污染权"；如果工厂有权排污（Property Rule 2），农民可以付钱让工厂少排。两种情况下，**总剩余最大的排污水平** 都会实现——只是分配不同。

**三个关键假设**：
1. 交易成本为零
2. 产权清晰
3. 信息对称

**现实 = 打破这些假设**：交易成本高（大量受害者、谈判困难），所以需要法律。

### 9.3 法律工具对比

| 工具 | 何时用 |
|---|---|
| **私人谈判** | 少数人、产权清晰、低交易成本 |
| **侵权责任** | 多数人受害、有因果关系可证 |
| **监管 (command and control)** | 技术要求可明确、执行简单 |
| **税收 (Pigouvian tax)** | 边际成本 vs 收益平衡 |
| **可交易许可证 (cap and trade)** | 总量已定，希望最便宜地达成 |

**Pigouvian Tax**：理想上，税率 = 边际外部成本。这样私人决策者就会把外部成本内部化。

### 9.4 Coase-Calabresi 财产规则 vs 责任规则

详见 `economic-analysis.md`。核心洞察：
- **财产规则 (Property Rule)**：禁令保护——只能通过 **自愿交易** 转让
- **责任规则 (Liability Rule)**：事后损害赔偿——任何人可以 **强制购买** 通过损害
- 选择取决于 **交易成本** 和 **估值信息** 谁更准确

## 10. 公共物品 Public Goods

### 10.1 两个特征

- **非排他性 (non-excludable)**：无法阻止非付费者使用
- **非竞争性 (non-rival)**：一个人使用不减少他人可用量

### 10.2 四象限分类

|  | 可排他 | 不可排他 |
|---|---|---|
| **竞争** | 私人物品（汉堡） | 公共资源（渔场）|
| **不竞争** | 俱乐部物品（有线电视） | 纯公共物品（国防） |

### 10.3 问题：搭便车与供给不足

私人不会生产公共物品（收不到钱），所以需要：
- **政府提供**（税收融资）
- **法律创造排他性**（专利制度让知识品变成俱乐部物品）
- **非市场机制**（捐赠、志愿）

### 10.4 公共资源悲剧 Tragedy of the Commons (Hardin, 1968)

**可竞争但不可排他** 的资源（公海渔场、草原、大气层）容易被过度使用。解决：
- **产权化**（私有化或集体产权）
- **配额**
- **Pigouvian 税**
- **社群自治**（Elinor Ostrom 的工作，2009 诺贝尔奖）

## 11. 福利经济学 Welfare Economics

### 11.1 帕累托效率 Pareto Efficiency

**定义**：没有任何改变能在不损害任何人的情况下改善某人的福利。

**限制**：几乎任何政策都有输赢家，所以严格的帕累托改进很稀缺。

### 11.2 Kaldor-Hicks 效率

**放松标准**：如果赢家能 **假想地** 补偿输家并仍然得益，那就是 Kaldor-Hicks 改进。这是 **法经济学中最常用的效率标准**——但隐藏了分配正义的问题。

### 11.3 功利主义 Utilitarianism

最大化 **总效用**（Bentham, Sidgwick, Mill）：

$$\text{Social Welfare} = \sum_i u_i(w_i)$$

问题：如何加总效用？为了 100 人的小福利牺牲 1 人的大福利，直觉上有问题。

### 11.4 Rawlsian Maximin

最大化 **最差处境者** 的福利（Rawls, *A Theory of Justice*, 1971）：

$$\text{max} \min_i u_i$$

强烈的再分配倾向，但也会损害激励。

### 11.5 再分配的成本

**效率与公平的权衡**：税收影响劳动供给（所得税使工作的边际回报降低）。

**Okun 的漏桶**：从富人那里拿 $\$100$，因为征税和发放的效率损失，只有 $\$60$–$\$80$ 到达穷人。我们愿意接受多大的漏桶？

## 13. Worked Example: 合并审查

两家相关公司市场份额分别为 $30\%$ 和 $20\%$，合并前市场 HHI = $2{,}500$。其他公司份额分别为 $20, 15, 10, 5$。

**合并前**：
$$HHI_{\text{pre}} = 30^2 + 20^2 + 20^2 + 15^2 + 10^2 + 5^2 = 900 + 400 + 400 + 225 + 100 + 25 = 2{,}050$$

**合并后**（两家合并为一家 $50\%$）：
$$HHI_{\text{post}} = 50^2 + 20^2 + 15^2 + 10^2 + 5^2 = 2{,}500 + 400 + 225 + 100 + 25 = 3{,}250$$

**变化**：$\Delta HHI = 3{,}250 - 2{,}050 = 1{,}200$

**判断**：post-HHI > $1{,}800$（高度集中），$\Delta HHI > 200$ → 根据 2023 Merger Guidelines 被 **推定** 具有反竞争效应，需要被合并方拿出充分的 **效率抗辩 (efficiency defense)** 或 **出售资产 (divestiture)** 方案。

**快速验证**：合并企业双方的 **交叉价格弹性** 是多少？如果高（很接近的替代品），涨价损害就大；如果低，反而可能没什么问题。

## 14. 律师的微观经济学对抗清单

### 面对反垄断指控 / 辩护

1. **相关市场怎么定义**？用 SSNIP 还是其他方法？
2. **市场份额计算** 的依据是什么？销售额 vs 销量 vs 产能？
3. **HHI 变化** 超过阈值了吗？
4. **是否有效率抗辩**？规模经济、R&D 整合？
5. **入口障碍** 有多高？
6. **其他竞争者的反应** 如何？
7. **消费者证据**：他们的交叉弹性、转换成本

### 面对市场行为分析

8. **需求弹性** 估计合理吗？数据来源？
9. **成本度量** 用的是什么？AVC？AIC？LRMC？
10. **因果关系**：价格变化是因为被告行为还是外部冲击？

## 16. 常见陷阱速查表

| 陷阱 | 症状 | 修正 |
|---|---|---|
| 市场界定过宽 / 过窄 | 份额随意 | 用 SSNIP |
| 毛利率 = 垄断 | 直接上结论 | 算 Lerner，考虑效率 |
| HHI 计算错误 | 份额基础不对 | 用销售额，核对所有公司 |
| 忽略进入壁垒 | 静态分析 | 动态、长期视角 |
| 忽略 **效率** 抗辩 | 只看 DWL | 考虑规模经济 |
| Coase 乱用 | 忽略交易成本 | 现实中 TC 几乎都很高 |
| 误判弹性 | 短期推长期 | 长期弹性 > 短期 |
| 忽视产品差异化 | 把不同产品算一个市场 | 区分 tier 和定位 |
| 混淆名义共谋与默契共谋 | 只找协议证据 | 默契也可以反竞争 |
