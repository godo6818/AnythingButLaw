# 法经济学 Economic Analysis of Law

> 法经济学不是"用经济学帮法律客户打官司"，而是 **用经济学思维分析法律制度本身**。为什么合同法这样设计？为什么侵权法既有严格责任又有过错责任？刑罚的最优力度是多少？这一章把法律制度视为一套 **激励系统** 来研究。这是美国法学院的核心课程之一，Tracy Wang 相信每个中国律师都应该学。

## 2. 合同法的经济分析

### 2.1 完全合同与不完全合同

**完全合同 (completely specified contract)**：对所有可能的未来状态都规定了双方的行动。**现实中不存在**。

**为什么不完全**：
1. **前瞻成本**：列举所有状态太贵
2. **描述成本**：即使想到也难用法律语言描述
3. **验证成本**：即使描述了，法院也难核实
4. **认知限制**：人想不到所有可能性

### 2.2 法院解释合同的经济学角色

当合同没有明确规定时，法院填补的 **缺省规则 (default rules)** 实际上是在为所有类似合同提供 **隐含条款**。

**两种思路**：

**多数派缺省 (majoritarian default)**：设定"如果当事人事先考虑过，大多数人会选择的条款"。**省钱**，让当事人免于重复起草。

**惩罚性缺省 (penalty default, Ayres-Gertner 1989)**：故意设置一个 **不合双方意愿** 的缺省，诱使当事人 **显式表达** 真实意图。特别适合知情方需要向不知情方披露的情境。

**例**：*Hadley v. Baxendale* (1854) 的 "foreseeability rule"——只赔偿可预见的损失。这迫使原告（知情方）事先披露特殊情况，否则风险自担。这是经典的 penalty default。

### 2.3 违约损害赔偿 Damages for Breach

| 类型 | 含义 | 效率含义 |
|---|---|---|
| **期望损害 Expectation** | 让受害方处于合同被履行时的状态 | **效率违约的理论基础** |
| **信赖损害 Reliance** | 补偿受害方因信赖合同而发生的支出 | 较保守，可能抑制效率违约 |
| **恢复原状 Restitution** | 返还违约方从合同获得的利益 | 最低限度救济 |
| **约定损害 Liquidated** | 双方预先约定的赔偿金额 | 效率，但受"罚金" doctrine 限制 |

### 2.4 Efficient Breach 效率违约

**核心命题**：当违约 + 赔偿期望损害总体比履行更有效率时，**违约是社会最优的**。

**Worked Example**：

卖方承诺以 $\$100$ 向 A 供货。A 的使用价值 $\$120$（所以 A 的 surplus = $\$20$）。

**突发事件**：出现买方 B 愿付 $\$150$。

**如果强制履行**：卖方给 A，总剩余 = $\$120 - \$100 = \$20$（A 的使用价值是货物归宿）。

**如果允许违约 + 期望损害赔偿**：
- 卖方违约，给 A 付 $\$20$（期望损害，恢复 A 的 expectation）
- 卖方卖给 B，收 $\$150$，净收益 $\$150 - \$100 - \$20 = \$30$
- B 获得 $\$150$ 价值的货物
- 总剩余 = $(A 拿到 \$20) + (卖方赚 \$30) + (B 获得 \$150 - \$150 = \$0) = \$50$

**效率改进**：总剩余从 $\$20$ 增加到 $\$50$。这就是为什么英美法默认允许违约 + 金钱赔偿（而大陆法更强调 specific performance）。

### 2.5 效率违约的质疑

- **道德主义批评**：违约即使"有效率"也是违背承诺，应该强制履行
- **谈判可能性**：A 也许本来就会同意让路给 B，只要价格合适——这回到 Coase 定理，关键是 **交易成本**
- **惩罚性缺省反例**：如果违约太容易，合同的事前价值下降

### 2.6 实际履行 Specific Performance

**英美法**：默认 **不给** specific performance——只有在金钱赔偿不充分时（独一无二的艺术品、不动产）才使用。

**大陆法**：默认 **给** specific performance。

**经济学分析**：
- 大陆法更好保护 expectation interest，不允许效率违约
- 英美法更灵活，允许效率重新分配
- 两种制度下 **最终 outcome 可能一样**（Coase 定理），但分配和交易成本不同

## 3. 侵权法的经济分析

### 3.1 Hand 公式 (United States v. Carroll Towing, 1947)

Learned Hand 法官的经典公式定义了过失 (negligence)：

$$\text{有过失 if} \quad B < P \cdot L$$

- $B$ = 防范措施的 **成本 (burden)**
- $P$ = 事故 **发生概率 (probability)**
- $L$ = 事故 **损失大小 (loss)**

**现代形式**：边际 Hand 公式——只要 **边际防范成本 < 边际预期损失减少**，就应该继续投入防范。

**Worked Example**：

一家工厂排放的化学品有 $P = 0.01$（每年 $1\%$）概率污染下游水源，损失 $L = \$10M$。安装过滤设备成本 $B = \$80{,}000$。

$$P \cdot L = 0.01 \times 10{,}000{,}000 = \$100{,}000 > \$80{,}000 = B$$

所以 **没装过滤 = 过失**。

**反过来**：如果过滤设备成本 $\$120{,}000$，$B > PL$，不装过滤就不是过失——社会效率最优的是 **承担预期损失而非防范**。

### 3.2 严格责任 vs 过错责任

| 责任形式 | 定义 | 激励效果 |
|---|---|---|
| **严格责任 Strict Liability** | 造成损害就赔，不问过错 | 加害人 **内部化全部** 外部成本 → 最优防范 + 最优活动水平 |
| **过错责任 Negligence** | 只有未达到合理防范标准才赔 | 加害人达到标准后 **不承担边际损失** → 最优防范但 **过度活动** |

**关键洞察 (Shavell)**：
- 两种形式下，**防范水平** 都最优（加害人会选 $B = PL$）
- 但 **活动水平** 不同：严格责任下加害人会减少危险活动，过错责任下会过度
- 严格责任适合 **高风险、活动水平关键** 的情境（wild animals, ultrahazardous activities）
- 过错责任适合 **活动水平本身有价值** 的情境（driving）

### 3.3 双边预防 Bilateral Care

如果 **双方都能影响事故概率**（加害人的谨慎 + 受害人的谨慎），单方的严格责任就不够——需要 **contributory negligence** 或 **comparative negligence** 作为补充。

### 3.4 Calabresi-Melamed 框架 (1972)

**"One View of the Cathedral"**：根据 **交易成本** 和 **估值信息**，选择不同的法律保护方式：

| | 财产规则 (Property Rule) | 责任规则 (Liability Rule) |
|---|---|---|
| **保护权利人** | 未经同意不得取走——需 **协商** | 可以取走 + 事后付损害赔偿 |
| **例** | 土地：未经同意不得占用 | 侵权：事后赔钱 |
| **何时最优** | 交易成本低、估值难定（情感价值） | 交易成本高、损害可估 |

**四种基本组合**：
1. **Rule 1**：权利人受财产规则保护（最强）
2. **Rule 2**：权利人受责任规则保护——任何人可以"购买"通过损害赔偿
3. **Rule 3**：另一方受财产规则保护
4. **Rule 4**：另一方受责任规则保护（反向禁令——原告必须付钱让被告停止，罕见）

**经典 Spur Industries v. Del E. Webb (1972)** 案就是 Rule 4 的应用：开发商必须付钱让养牛场搬走，因为开发商"买下"了污染权。

## 4. 犯罪与刑罚的经济学

### 4.1 Becker (1968) 的理性犯罪模型

犯罪者是 **理性的**，犯罪 iff：

$$\text{预期犯罪收益} > p \cdot s + \text{合法收益}$$

其中 $p$ = 被抓概率，$s$ = 制裁力度（monetary equivalent）。

**所以**：社会可以通过增加 $p$ 或 $s$ 来减少犯罪。哪个更便宜？

### 4.2 最优制裁理论

**Becker-Polinsky-Shavell** 的核心洞察：

**如果执法成本为零**，最优策略是 **极高的制裁 + 很低的被抓概率**（威慑效果相同，但执法成本最小）。

$$p \cdot s = \text{const}$$

**例**：如果 $\$1M$ 罚金 + $1\%$ 被抓，和 $\$10K$ 罚金 + $100\%$ 被抓，威慑相同，但后者的执法成本高得多。

**为什么现实中不这样**？
1. **有限财富**：罚金不能超过犯罪者的资产
2. **风险规避**：高制裁引发过度威慑
3. **错判的代价**：概率极低时，错判一次的代价高
4. **边际威慑 (marginal deterrence)**：如果抢劫和谋杀都判死刑，抢劫犯就会顺手杀人（已经到顶了）

### 4.3 为什么关注意图 (mens rea)？

**经济学解释**：对 **有意加害** 者施加更高制裁，因为：
- 他们 **更难威慑**（从犯罪中获益更大）
- **更可能隐瞒** 行为（$p$ 更低）
- **更可能造成大伤害**
- **更有可能重犯**

所以 mens rea 的分层（intentional > reckless > negligent > strict liability）不是纯粹的道德要求，而是有效率基础。

### 4.4 未遂罪 Punishment of Attempts

**经济学角度**：惩罚未遂 = 增加 **被抓概率 $p$** 的一种方式（抓到未遂比抓到既遂更早）。理想上未遂的制裁应该 = 既遂的制裁的 **期望值折扣**。

### 4.5 监禁 Incapacitation

监禁是最昂贵的制裁方式（社会成本 = 坐牢者的生产力损失 + 监狱运营成本）。

**只有在以下条件同时成立时才最优**：
1. 罚金不足以威慑（犯罪者没钱）
2. 预期未来伤害 > 监禁成本
3. 其他非监禁手段（社区服务、吊销执照）不够

### 4.6 死刑与威慑

**实证争议**：死刑是否比终身监禁更能威慑谋杀？数据 **不一致**。

**经济学分析**：如果终身监禁已经是极高的期望成本，**边际威慑** 效应接近零——因为"再多一点"对理性犯罪者已经没有额外威慑力。

## 5. 民事诉讼的经济学

### 5.1 起诉决策

当事人会一直投入诉讼成本，只要 **边际成本 < 边际预期收益**。

**理性起诉条件**：

$$P_{\text{win}} \cdot E[\text{判决}] - C_{\text{诉讼成本}} > 0$$

### 5.2 和解 vs 审判

**为什么大多数案件和解**？因为 **总蛋糕大**：双方避免诉讼成本。和解空间：

$$\text{ZOPA} = P_{\text{原告估计}} \cdot J - C_P - (P_{\text{被告估计}} \cdot J + C_D)$$

$$= (P_{\text{原告}} - P_{\text{被告}}) \cdot J - (C_P + C_D) \cdot (-1)$$

等等，让我重写。和解有空间的条件：

$$(P_{\text{原告}} \cdot J - C_P) < (P_{\text{被告}} \cdot J + C_D)$$

即 **原告的审判期望收入 < 被告的审判期望成本**。

变形：

$$P_{\text{原告}} - P_{\text{被告}} < \frac{C_P + C_D}{J}$$

**解读**：只要双方对胜诉概率的分歧 **小于** 诉讼总成本除以判决金额，就存在和解空间。

**和解增加的条件**：
1. 双方概率判断相似（没有过度乐观偏误）
2. 风险规避（每一方都愿意为 **确定性** 付溢价）
3. 诉讼成本高（$C_P + C_D$ 大）
4. 判决金额小（$J$ 小）

**审判发生的条件**：
1. 双方判断严重分歧（双方都觉得自己会赢）
2. 风险偏好（风险规避反方向）
3. 非金钱考虑（声誉、先例、惩罚对方）

### 5.3 Priest-Klein 假说 (1984)

**选择偏差**：进入审判的案件 **不代表** 所有争议。和解会过滤掉"结果明显"的案件，**留下来审判的都是"五五开"的**——这解释了为什么进入审判的案件胜率接近 $50\%$。

**律师含义**：不要用"历史审判胜率"来估计你案子的胜率——因为那个池子是 **选择后的**。用类似案件的 **和解率 + 平均和解金额** 作为基准。

### 5.4 诉讼费分担规则

- **美国规则 (American Rule)**：各自承担律师费
- **英国规则 (English Rule)**：败诉方承担对方律师费

**经济学比较**：
- **美国规则** 鼓励小额诉讼（败诉风险可控）→ 更多"滋扰性诉讼"
- **英国规则** 抑制弱案件 → 但对**穷人不利**（风险厌恶放大）
- 各有利弊，没有理论上的绝对优劣

## 6. 法律的正当性：什么时候应该推翻私人意思自治？

### 6.1 外部性存在时

- 合同损害第三方福利 → 强制法律规范
- 例：不得约定垄断（反垄断）、环境合规

### 6.2 信息严重不对称时

- 强制披露义务（证券法、消费者保护法）
- 冷静期 (cooling-off period)

### 6.3 认知限制 / 行为经济学

- **损失厌恶、现时偏差、过度自信** → 人们会做后悔的决定
- **家长主义 (paternalism)**：强制退休储蓄、强制保险
- **Libertarian paternalism (Sunstein-Thaler)**：默认选项设计

### 6.4 不可让渡之物 (Inalienable)

- 人体器官、婴儿、投票权、自我奴役
- 理由：**保护人格尊严**、**防止胁迫**、**市场侵蚀非市场领域**

## 7. 法律设计的两个基本选择

### 7.1 规则 vs 标准 Rules vs Standards

- **规则 (rules)**：事前明确（"限速 60 km/h"）
  - 优点：可预见、易执行、减少套利
  - 缺点：过度包容或不足包容
- **标准 (standards)**：事后判断（"合理驾驶速度"）
  - 优点：灵活、公平
  - 缺点：不确定性、执行成本高

**Kaplow (1992)** 的经济分析：当行为 **频繁发生** 且 **变化不大**，用规则；当行为 **少见** 但 **情境复杂**，用标准。

**例**：交通速限用规则（每天发生），证券欺诈用标准（每个案件都不同）。

### 7.2 事前 vs 事后 Ex Ante vs Ex Post

- **事前规范**：制定规则让行为合规（监管）
- **事后救济**：等到损害发生再补救（侵权）

**选择取决于**：
- 是否能提前识别违规（事前规范需要）
- 执法成本（事前监管 vs 事后诉讼哪个便宜）
- 错误的代价（事前可能过度限制，事后可能不可逆）

## 8. 管制的经济学

### 8.1 为什么管制？（市场失灵）

- 垄断
- 外部性
- 信息不对称
- 公共物品

### 8.2 管制的成本

**Stigler (1971) 的管制俘获理论**：管制机构倾向于 **被被管制行业俘获**——因为：
- 行业有集中利益（每家公司损失大）
- 公众有分散利益（每个消费者损失小）
- 行业有专业知识
- 旋转门现象

**律师实务含义**：挑战监管规则时，经济学论据往往是 **"这个规则真的服务公共利益，还是保护现有企业排斥竞争？"**

### 8.3 Cost-Benefit Analysis (CBA)

行政机构的经济学标准工具：

$$NPV(\text{规制}) = \sum_{t} \frac{\text{社会收益}_t - \text{合规成本}_t}{(1 + r)^t}$$

**争议**：
- 如何量化生命的价值？($$11M$$ VSL 是 EPA 的标准，但道德争议大)
- 折现率的选择（气候变化争议的核心）
- 哪些收益 / 成本可量化？非量化的怎么办？

## 10. 律师的法经济学对抗清单

### 在辩护 / 起诉中使用经济学论据

1. **Hand 公式**：被告的防范成本 vs 预期损失，给陪审团一个可计算的框架
2. **Coase 定理**：挑战强制性规则——"这个问题应该通过私人协商解决吗？"
3. **效率违约**：支持金钱赔偿而非强制履行
4. **Penalty default**：论证某个缺省规则是为了诱使披露
5. **Priest-Klein**：不要被"历史胜率"吓到——进入审判的案子都是分歧大的

### 在政策辩论 / 监管听证中

6. **CBA 分析**：支持或反对新规
7. **管制俘获论据**：质疑"保护消费者"规则是否实际保护产业
8. **规则 vs 标准**：根据行为频率选择合适形式
9. **边际威慑**：挑战一刀切的严格制裁

| 陷阱 | 症状 | 修正 |
|---|---|---|
| 效率 = 公平 | 把 Kaldor-Hicks 当正义 | 分离效率和分配问题 |
| 忽略交易成本 | 乱用 Coase | 现实 TC 往往很高 |
| 单边预防错误 | 只让一方预防 | 双边 → contributory negligence |
| 忽略活动水平 | 只看防范水平 | 严格责任 vs 过错影响活动 |
| 边际威慑失败 | 一刀切严刑 | 保留制裁梯度 |
| Becker 的僵化使用 | 假设犯罪者完全理性 | 考虑行为经济学限制 |
| 混淆 penalty 和 majoritarian default | 不清楚设计意图 | 明确目标再选缺省 |
| 忽略管制俘获 | 以为监管就是公益 | 用利益集团分析 |
| 误用 Priest-Klein | 用胜率预测个案 | 只适用于"审判样本" |
