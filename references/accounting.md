# 会计 Accounting

> 律师不需要做账，但需要 **读懂账**。在尽调中发现问题、在诉讼中质疑财务数据、在谈判中判断对方的财务状况、在 M&A 中识别潜在地雷——这些都需要会计基础。这一章目的不是让你去考 CPA，而是让你能和 CFO 对话、能识别红旗、能问出正确的问题。

## 1. 三大财务报表 The Big Three

### 1.1 资产负债表 Balance Sheet (Statement of Financial Position)

**某一时刻的财务快照**。通常展示两个时间点（期初和期末）以便比较。

**会计恒等式**：

$$\text{资产 Assets} = \text{负债 Liabilities} + \text{所有者权益 Equity}$$

或者写成：

$$\text{Equity} = \text{Assets} - \text{Liabilities}$$

**结构示例**（简化）：

```
资产                            负债
  流动资产                         流动负债
    现金 & 等价物                    应付账款 AP
    应收账款 AR                      短期借款
    存货                            其他流动负债
    预付费用                       流动负债合计
  流动资产合计                    长期负债
  非流动资产                        长期借款
    PP&E（固定资产）                应付债券
    无形资产                        递延所得税
    商誉                            其他长期负债
    长期投资                      长期负债合计
  非流动资产合计                  负债合计
总资产                          所有者权益
                                  股本
                                  资本公积
                                  留存收益
                                  累计其他综合收益
                                权益合计
                                负债 + 权益
```

**关键局限**：
- **某些重要资产不在表上**——品牌价值、人才、客户关系、未确认的知识产权、未并表的 JV
- **历史成本可能严重脱离当前价值**——10 年前 $100M 买的地，现在可能值 $1B
- **商誉 (goodwill)** 不摊销只减值——一旦减值常常意味着当年的并购决策错了

### 1.2 利润表 Income Statement

一段时期内的经营成果。

```
销售收入 Sales / Revenue
- 销售成本 COGS (Cost of Goods Sold)
= 毛利 Gross Margin
- 营业费用 Operating Expenses (SG&A, R&D, 折旧摊销)
= 营业利润 Operating Income / EBIT
± 非营业项目 (利息、投资收益、资产处置)
= 税前利润 EBT (Income Before Taxes)
- 所得税 Income Tax
= 净利润 Net Income
```

**关键概念**：
- **EBIT (Earnings Before Interest and Taxes)**：资本结构中立的盈利能力
- **EBITDA**：EBIT 加回折旧摊销——常用于比较资本密集行业，但 Warren Buffett 说过"EBITDA 就是把资本支出当成不存在"
- **Operating Income vs Net Income**：营业利润反映核心业务，净利润还包括一次性项目——分析师关注 **持续性盈利 (core earnings)**

### 1.3 现金流量表 Statement of Cash Flows

反映一段时期内现金的 **来源** 和 **去向**，揭示企业的 **流动性 (liquidity)**——按时支付款项的能力。

**三大类别**：

| 类别 | 内容 | 关注点 |
|---|---|---|
| **经营活动 OCF** | 日常业务产生 / 消耗的现金 | 是否持续为正？vs 净利润？ |
| **投资活动 ICF** | 购买 / 出售长期资产、投资 | 资本支出规模？是否匹配业务增长？ |
| **筹资活动 FCF** | 借贷、发行 / 回购股票、分红 | 依赖外部融资？分红稳定性？ |

**两种编制方法**：
- **直接法 (direct method)**：列示各类现金收入和支出——信息更丰富但编制贵
- **间接法 (indirect method)**：从净利润出发调整非现金项目——最常见

**间接法的调整**：

```
净利润
+ 折旧与摊销（非现金费用）
± 营运资本变化
    - 应收账款增加（赚了但没收到现金）
    + 应付账款增加（欠款但没支付）
    - 存货增加（买了但没卖）
+ 其他非现金项目（股权激励、减值损失）
= 经营活动现金流
```

### 1.4 关键教训：**净利润 ≠ 现金流**

一家公司可以 **账面盈利但现金枯竭**：
- 收入确认后，客户却不付款 → 应收账款膨胀
- 为了扩张囤积存货 → 存货膨胀
- 资本支出巨大但折旧平滑 → 账面利润好看但现金出血

**律师的一个红旗信号**：**净利润连年上升，但经营现金流连年下降或为负**。这是收入确认激进或应收账款失控的典型症状，也是大多数财务舞弊案的起点。

## 2. 复式记账法 Double-Entry Bookkeeping

每笔交易同时影响 **至少两个账户**：

| | 增加 | 减少 |
|---|---|---|
| **资产** | 借方 (Debit, Dr) | 贷方 (Credit, Cr) |
| **负债** | 贷方 (Credit) | 借方 (Debit) |
| **权益** | 贷方 (Credit) | 借方 (Debit) |
| **收入** | 贷方 (Credit) | 借方 (Debit) |
| **费用** | 借方 (Debit) | 贷方 (Credit) |

**记账循环**：
1. 交易发生 → 在 **日记账 (journal)** 中记录
2. 过账到 **总账 / 分类账 (ledger)**
3. 期末编制 **试算平衡表 (trial balance)**
4. 期末调整分录（权责发生制调整）
5. 关闭临时账户（收入、费用），转入 **留存收益**
6. 编制财务报表

### 2.1 一个 Worked Example

一家公司发行 $\$1M$ 股票融资，然后买了一台 $\$400K$ 的设备。

**分录 1**（发行股票）：
```
Dr  现金            $1,000,000
    Cr  股本                    $1,000,000
```

**分录 2**（购买设备）：
```
Dr  PP&E（设备）     $400,000
    Cr  现金                    $400,000
```

**报表变化**：
- 现金 $0 → +$600K（+$1M$ − $400K$）
- PP&E $0 → +$400K$
- 股本 $0 → +$1M$
- 总资产 $0 → +$1M$；总负债 + 权益 $0 → +$1M$ ✓ 平衡

## 3. 核心会计原则

### 3.1 历史成本 Historic Cost
以购买时实际支付的金额记录。**可靠 (reliable)** 但不一定 **相关 (relevant)**。可靠和相关的矛盾是会计学的永恒主题。

### 3.2 配比原则 Matching Principle
收入归属于"为产生该收入而付出努力"的期间；费用归属于"其带来收益"的期间。这是权责发生制的灵魂。

### 3.3 资本化与折旧 Capitalization & Depreciation

**资本化**：某些支出不作为当期费用，而是作为资产放在资产负债表上，在其有用寿命内逐步折旧。

**折旧方法**：
- **直线法 (straight-line)**：$\text{每期折旧} = \dfrac{\text{成本} - \text{残值}}{\text{使用年限}}$
- **加速折旧 (accelerated)**：前期折旧更多（双倍余额递减、年数总和法）
- **产量法 (units of production)**：按实际使用量

**舞弊点**：故意延长折旧年限 → 当期折旧减少 → 利润虚增。Waste Management（1998）就是经典案例——把垃圾车折旧年限从 10 年改到 18 年，虚增利润超过 $\$1B$。

### 3.4 权责发生制 vs 现金制

**权责发生制 (accrual basis)**：收入在 **赚取** 时确认（不论现金是否收到），费用在 **发生** 时确认。这是 GAAP/IFRS 的默认。

**现金制 (cash basis)**：只有现金收付时才记账。小企业可能用，但不符合 GAAP。

**应计 (accrual)** vs **递延 (deferral)**：
- **应计**：已赚取但未收现 → 应收账款；已发生但未支付 → 应付账款 / 应付费用
- **递延**：已收现但未赚取 → 预收账款；已支付但未消耗 → 预付费用

### 3.5 收入确认 Revenue Recognition

**ASC 606 / IFRS 15 的五步法**（2018 年起全球统一）：
1. 识别与客户的合同
2. 识别履约义务
3. 确定交易价格
4. 把交易价格分配到各履约义务
5. 当履约义务被履行时确认收入

**律师必问**：
- **何时** 履行被认为完成？发货？客户验收？90 天退货期过后？
- **可变对价 (variable consideration)** 怎么估计？——bill-and-hold 交易、channel stuffing 常发生在这里
- 是否存在 **right of return**？返回预期如何估计？

**这是财务舞弊最常见的操纵点**——大约 60% 的 SEC 舞弊执法案件涉及收入确认。

## 4. 存货计价方法

| 方法 | 特点 | 利润表 | 资产负债表 |
|---|---|---|---|
| **FIFO (先进先出)** | 先买的先卖 | 成本反映较早价格 → 通胀时利润偏高 | 存货接近当前成本，较准确 |
| **LIFO (后进先出)** | 后买的先卖 | 成本反映最近价格 → 通胀时利润偏低（税盾效应）| 存货可能严重低估 |
| **加权平均** | 平均成本 | 折中 | 折中 |
| **个别认定** | 每件单独跟踪 | 最准确但最贵 | 最准确 |

**权衡**：利润表准确性 vs 资产负债表准确性 **不可兼得**。

**LIFO 的隐藏陷阱**：如果一家用 LIFO 的公司突然 **减少存货**（卖出多年前低成本买的存货），会触发 **LIFO liquidation**——老的低成本被释放到 COGS，当期利润暴涨，但这不代表经营改善。IRS 要求用 LIFO 报税的公司也要在报表中用 LIFO（LIFO conformity rule），美国之外几乎不允许 LIFO（IFRS 禁止）。

## 5. 公允价值 vs 历史成本

| | 公允价值 Fair Value | 历史成本 Historic Cost |
|---|---|---|
| **优势** | 更及时、更相关 | 更客观、更可靠 |
| **劣势** | 主观性强、易被操纵 | 可能严重偏离真实价值 |

**公允价值的三个层级 (ASC 820)**：
- **Level 1**：活跃市场的 **同一资产** 报价（最可靠）
- **Level 2**：可观察的市场数据（类似资产价格、利率曲线、收益率、波动率）
- **Level 3**：**不可观察的输入**——基于模型和主观假设

**律师红旗**：Level 3 资产占比高 → 估值可靠性低 → 操纵空间大。Enron、次贷危机中很多问题都出在 Level 3。

| 文件 | 内容 |
|---|---|
| **10-K** | 年报，经审计 |
| **10-Q** | 季报 |
| **DEF-14A** | 委托声明书（proxy）——含高管薪酬 |

### 6.3 审计意见等级

- **无保留意见 (unqualified / clean opinion)**：合理保证 (reasonable assurance)
- **保留意见 (qualified opinion)**：除某个具体问题外其他公允
- **否定意见 (adverse opinion)**：报表严重失真——几乎没人愿意发这个
- **无法表示意见 (disclaimer)**：审计师无法获得足够证据
- **持续经营疑虑 (going concern qualification)**：审计师对公司能否继续经营表示怀疑——**极强的红旗**

## 7. 财务比率分析 Financial Ratio Analysis

### 7.1 盈利能力 Profitability

$$\text{毛利率 Gross Margin} = \frac{\text{毛利}}{\text{收入}}$$

$$\text{营业利润率 Operating Margin} = \frac{\text{EBIT}}{\text{收入}}$$

$$\text{净利率 Net Margin} = \frac{\text{净利润}}{\text{收入}}$$

$$\text{ROA} = \frac{\text{净利润}}{\text{总资产}}$$

$$\text{ROE} = \frac{\text{净利润}}{\text{股东权益}}$$

### 7.2 DuPont 分解 The DuPont Identity

ROE 可以分解为三个驱动因素：

$$\text{ROE} = \underbrace{\frac{\text{净利润}}{\text{收入}}}_{\text{净利率}} \times \underbrace{\frac{\text{收入}}{\text{总资产}}}_{\text{资产周转率}} \times \underbrace{\frac{\text{总资产}}{\text{权益}}}_{\text{权益乘数}}$$

$$\text{ROE} = \text{Profit Margin} \times \text{Asset Turnover} \times \text{Financial Leverage}$$

**律师洞察**：两家 ROE 都是 $20\%$ 的公司可以完全不同：
- A：高利润率 + 低周转 + 低杠杆（奢侈品品牌）
- B：低利润率 + 高周转 + 低杠杆（沃尔玛）
- C：低利润率 + 低周转 + **高杠杆**（银行、金融公司）——**风险最高**

**ROE 突然上升**，永远要问"来源是利润率、周转、还是杠杆？"——高杠杆驱动的 ROE 提升常常是 **风险隐患**。

### 7.3 流动性 Liquidity

$$\text{流动比率 Current Ratio} = \frac{\text{流动资产}}{\text{流动负债}}$$

$$\text{速动比率 Quick Ratio} = \frac{\text{流动资产} - \text{存货}}{\text{流动负债}}$$

（速动比率更严格，因为存货可能难以迅速变现）

$$\text{现金比率} = \frac{\text{现金 + 等价物}}{\text{流动负债}}$$

**经验值**：
- 流动比率 $< 1.0$：短期偿债困难
- 流动比率 $1.5$–$2.0$：相对健康
- 流动比率 $> 3.0$：现金闲置，效率低

### 7.4 杠杆 Leverage

$$\text{资产负债率} = \frac{\text{总负债}}{\text{总资产}}$$

$$\text{D/E 比率} = \frac{\text{总负债}}{\text{权益}}$$

$$\text{利息保障倍数 Interest Coverage} = \frac{\text{EBIT}}{\text{利息费用}}$$

利息保障倍数 $< 1.5$ 是破产风险的早期信号。

### 7.5 效率 Efficiency

$$\text{存货周转率} = \frac{\text{COGS}}{\text{平均存货}}$$

$$\text{应收账款周转天数 DSO} = \frac{\text{应收账款}}{\text{收入}} \times 365$$

**DSO 突然拉长**：可能是客户变差、可能是信用政策放松推销售、可能是收入确认虚假。

### 7.6 市场估值

$$\text{EPS} = \frac{\text{净利润} - \text{优先股股息}}{\text{加权平均流通股数}}$$

$$\text{P/E} = \frac{\text{股价}}{\text{EPS}}$$

$$\text{P/B} = \frac{\text{股价}}{\text{每股账面价值}}$$

$$\text{EV/EBITDA} = \frac{\text{企业价值}}{\text{EBITDA}}$$

其中 $\text{企业价值 (EV)} = \text{市值} + \text{总负债} - \text{现金}$。

## 8. 舞弊识别模型

### 8.1 Altman Z-Score（破产预测）

Edward Altman (1968) 开发的模型：

$$Z = 1.2 X_1 + 1.4 X_2 + 3.3 X_3 + 0.6 X_4 + 1.0 X_5$$

其中：
- $X_1 = $ 营运资本 / 总资产
- $X_2 = $ 留存收益 / 总资产
- $X_3 = $ 息税前利润 / 总资产
- $X_4 = $ 市值 / 总负债账面价值
- $X_5 = $ 销售额 / 总资产

**解读**：
- $Z > 2.99$：**安全区**
- $1.81 < Z < 2.99$：**灰色区**
- $Z < 1.81$：**危险区**，破产风险高

### 8.2 Beneish M-Score（盈余操纵预测）

Messod Beneish (1999) 开发，用于预测财务操纵。用 **8 个财务比率的变化** 计算 M-Score。

**阈值**：$M > -1.78$ → 可能存在盈余操纵的嫌疑。

**著名案例**：康奈尔 MBA 学生用 Beneish 模型在 Enron 破产前几个月就已经把它标记为 "manipulator"——学术界早就警告过。

### 8.3 红旗信号清单

1. **收入 $\uparrow$ 但经营现金流 $\downarrow$**：收入质量存疑
2. **应收账款增长远快于收入增长**：channel stuffing
3. **毛利率无法解释的突然提升**
4. **频繁变更会计政策或估计**
5. **审计师变更**（尤其是辞聘方向）
6. **复杂的关联方交易**
7. **Level 3 资产占比高且增长**
8. **商誉占总资产比例高且长期未减值**
9. **管理层激励严重挂钩短期财务指标**
10. **CFO / 审计委员会主席频繁变动**
11. **资本支出远低于折旧**（资产老化未替换）
12. **一次性项目频繁出现**（用来"正常化"收益）

## 9. 商誉与并购会计 Goodwill & M&A

**商誉 Goodwill** = 收购价格 − 被收购方 **可识别净资产的公允价值**。

这是"为品牌、客户关系、协同效应等无形资产多付的钱"。商誉 **不摊销**，但每年需要测试 **减值 (impairment)**。

**律师红旗**：
- 商誉占总资产 > 50%：过度依赖无形资产
- 从未计提减值的大额商誉：管理层可能在掩盖错误的并购
- 突然的大额减值：承认过去的决策失败

**AOL-Time Warner 2002**：$\$99B$ 商誉减值——历史上最大的一次，承认了 $\$164B$ 并购的价值几乎全部蒸发。

## 10. 律师实务场景

### 10.1 尽职调查 Due Diligence

**标准流程**：
1. 三年（最好 5 年）完整报表 + 审计报告
2. 计算所有关键比率的趋势
3. 与行业基准 / 可比公司对比
4. 关注红旗信号
5. 检查表外项目（经营租赁、或有负债、SPE）
6. 调查关联方交易
7. 与审计师沟通（如果允许）
8. 现场访谈 CFO、会计主管

### 10.2 诉讼中的财务分析

- **支付能力评估**：被告赢了能不能付？用流动性和杠杆比率
- **损害赔偿量化**：被告的利润结构（见 `finance.md` 的 lost profits 计算）
- **破产程序**：资产充足性、欺诈转移分析
- **证券欺诈**：比较事件前后的财报，识别何时开始失真

### 10.3 M&A 谈判

- **Earn-out 条款**：如果基于 EBITDA，买方要防止卖方 **操纵** 过渡期 EBITDA
- **净营运资本调整**：常被忽视但可能是百万级的调整
- **税务递延资产 (DTA)**：值多少钱取决于未来盈利能力
- **表外负债**：养老金缺口、环境责任、诉讼准备金

## 11. 律师的会计对抗清单

1. **现金流与净利润对比**：每年都对得上吗？
2. **主要应计项目趋势**：AR、存货、预付费用是否异常增长？
3. **会计政策变更**：改了什么？影响几何？
4. **关键估计**：折旧年限、坏账准备、存货跌价——是否激进？
5. **关联方交易**：金额、定价、披露是否充分？
6. **表外项目**：租赁（ASC 842 已并表）、担保、或有负债
7. **Level 3 资产**：比例、估值方法、假设
8. **商誉**：金额、来源、减值历史
9. **杠杆与流动性**：利息保障、流动比率、到期结构
10. **审计师独立性**：非审计服务费用占比？有无轮换？

## 12. 书中经典案例

- **Sterling Homex (1972)**：建设公司通过虚假收入确认虚增利润，最终破产，高管入狱——早期的收入舞弊案例
- **Enron (2001)**：SPE、mark-to-model、Level 3 估值操纵的教科书
- **WorldCom (2002)**：把 $\$3.8B$ 的运营费用资本化为资产——最简单也最致命的手法
- **Waste Management (1998)**：通过延长折旧年限虚增利润 $\$1.7B$
- **Lehman Brothers (2008)**：Repo 105 交易——季末暂时出表降低杠杆
- **Satyam (2009)**：印度 IT 巨头，伪造了 $\$1B$ 的现金存款

## 13. 常见陷阱速查表

| 陷阱 | 症状 | 解法 |
|---|---|---|
| 净利润导向 | "公司今年赚了 5 亿" | 看 OCF 和 FCF 是否一致 |
| 忽视表外项目 | "负债率不高啊" | 检查租赁、担保、SPE |
| 信任审计师 | "四大审过的" | 审计只是合理保证，不是绝对 |
| EBITDA 迷信 | "看 EBITDA 很好" | 算回折旧和资本支出 |
| 忽视会计政策变更 | 比较同比数据 | 调整到相同口径 |
| 商誉不减值就是好 | "账面看起来稳" | 真实业务表现如何？ |
| 只看比率不看趋势 | 单期数据 | 3–5 年趋势更重要 |
| 忽视行业差异 | 跨行业比率比较 | 同一行业内比较 |
