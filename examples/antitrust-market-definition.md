# 反垄断的市场界定：食品配送平台的合并审查

> **涉及知识域**：微观经济学、统计、博弈论

## 案情

2025 年初，美国两大食品配送平台 **ZipBite**（市占 $28\%$）和 **FlashFood**（市占 $22\%$）宣布合并，合并对价 $\$14B$。

美国司法部反垄断司 (DOJ) 发起 Second Request，开始 Hart-Scott-Rodino 审查。你是被合并方 ZipBite 的外部反垄断顾问，需要准备 **市场界定** 和 **效率抗辩** 的经济分析，以论证合并不会引起反竞争效应。

**核心问题**：
1. 相关市场是 **"食品配送"** 还是更宽？
2. 合并后的 HHI 是什么水平？
3. 合并是否会推高价格 / 减少选择？
4. 是否存在 **效率抗辩**？

---

## 分析

### 一、市场界定：SSNIP 测试

DOJ 标准方法：从最窄的候选市场开始，测试一个假想垄断者能否 **有利可图地涨价 $5\%$** 一年以上。

#### 候选市场 1：**只有食品配送**

**行业结构**：

| 公司 | 市占率 |
|---|---|
| MarketFood | $35\%$ |
| ZipBite | $28\%$ |
| FlashFood | $22\%$ |
| Grainhub | $10\%$ |
| 其他 | $5\%$ |

**HHI 合并前**：

$$HHI_{\text{pre}} = 35^2 + 28^2 + 22^2 + 10^2 + 5^2 = 1{,}225 + 784 + 484 + 100 + 25 = 2{,}618$$

**HHI 合并后**（合并 ZipBite + FlashFood = $50\%$）：

$$HHI_{\text{post}} = 35^2 + 50^2 + 10^2 + 5^2 = 1{,}225 + 2{,}500 + 100 + 25 = 3{,}850$$

**$\Delta HHI = 3{,}850 - 2{,}618 = 1{,}232$**

**按 2023 DOJ/FTC Merger Guidelines**：
- $HHI_{\text{post}} > 1{,}800$：**高度集中**
- $\Delta HHI > 200$ 且后集中度 > $1{,}800$：**推定反竞争**

**如果这个市场定义成立，合并几乎肯定被阻止**。

#### 候选市场 2：**所有"餐食获取"渠道**

扩大市场定义，包括：
- 堂食外卖 (dine-in + takeout)
- 自提 (pickup)
- 在家烹饪
- 企业餐饮服务 (meal kits)

**数据来源**：美国人每月餐食次数分布（来自 BLS Consumer Expenditure Survey）：

| 渠道 | 份额 |
|---|---|
| 在家烹饪 | $65\%$ |
| 堂食 | $15\%$ |
| 自提 | $10\%$ |
| 配送平台 | $8\%$ |
| Meal kits | $2\%$ |

如果用 $8\%$ 份额分子，合并后份额 = $50\% \times 8\% = 4\%$。**全市场 HHI 远低于阈值**。

**但这个定义成立吗**？SSNIP 测试登场。

### 二、SSNIP 检验

**假设**：ZipBite + FlashFood 的假想垄断者把配送平台的**价格**涨 $5\%$，**餐一次** 维持一年。会损失多少用户给其他渠道？

**关键概念：Critical Loss**

$$CL = \frac{\Delta P / P}{\Delta P / P + \text{贡献毛利率}}$$

假设配送平台的贡献毛利率 = $25\%$（行业研究数据），$\Delta P / P = 5\%$：

$$CL = \frac{0.05}{0.05 + 0.25} = \frac{0.05}{0.30} = 16.7\%$$

**所以**：涨价 $5\%$ 只要 **销量损失 < $16.7\%$** 就是有利可图的。

**实际损失率**（Actual Loss）估计：

通过 **diversion ratio** 分析——涨价 $5\%$ 时，用户会转向哪里？

消费者调查数据（ZipBite 内部调研，$n = 3{,}000$）：
- $44\%$ 会继续用（或忍受涨价）
- $18\%$ 转向其他配送平台（Grainhub, 等）
- $22\%$ 改为 **自提**
- $9\%$ 改为 **堂食**
- $7\%$ 改为 **在家烹饪**

**总离开率 = $56\%$**，其中：
- **只有 $18\%$ 留在"配送市场"内部**（转到 Grainhub）
- **$38\%$ 离开到其他渠道**

按候选市场 1（只配送）：
- 相关的损失只是离开配送平台市场的部分 = $38\%$
- $38\% \gg 16.7\%$（critical loss）
- **SSNIP 测试失败**！假想垄断者 **无法** 有利可图地涨价
- **结论**：候选市场 1 太窄，需要扩大

**扩大市场**，加入自提 $\to$ 重新做 SSNIP：

现在涨价后离开的损失只包括"完全退出餐食获取"的用户（$7\%$），$7\% < 16.7\%$，**SSNIP 通过**——这是相关市场的下界。

### 三、相关市场的"正确"定义

经济学家为原告（DOJ）做的回应：
- **消费者调查** 的数据可能被操纵（问题措辞、样本选择）
- **自提的转换成本**：下雨天、深夜、无车家庭 **无法** 转向自提
- **在家烹饪不是同类**：时间成本、技能、厨房设备

**原告的市场定义**：**"按需食品配送 (on-demand food delivery)"**，含义：
- 不含自提（因为需要交通）
- 不含订阅型 meal kit（时间错位）
- 不含堂食（完全不同的体验）

**被告（ZipBite 一方）的市场定义**：**"便利食品获取渠道"**，含义：
- 含所有快速获取现成食物的方式
- 含自提、在家烹饪部分情况（预制食品）

**这个争议决定了案件结果**。

### 四、单边效应 Unilateral Effects

即使市场被界定得窄，如果合并双方的产品 **不是彼此最近的替代品**，合并危害可能很小。

**Diversion Ratio 分析**：ZipBite 涨价时，流失用户中多少去了 FlashFood？

调查结果：ZipBite 涨价 $10\%$ 时：
- $20\%$ 去 MarketFood
- $35\%$ 去 FlashFood
- $15\%$ 去 Grainhub
- $30\%$ 退出

**Diversion ratio (ZipBite $\to$ FlashFood) = $35\%$**。

**这是相当高的值**——意味着 ZipBite 和 FlashFood 是 **彼此最近的替代品**。合并消除了这个竞争压力 → **单边效应显著**。

**UPP (Upward Pricing Pressure) 指标**：

$$UPP = D_{12} \cdot M_2 - E$$

- $D_{12}$ = ZipBite 到 FlashFood 的 diversion ratio = $0.35$
- $M_2$ = FlashFood 的毛利率 = $25\%$ 
- $E$ = 合并带来的边际成本节约

$$UPP = 0.35 \times 0.25 - E = 0.0875 - E$$

只要效率节约 $E < 8.75\%$，合并就产生 **上行定价压力**。

### 五、效率抗辩

合并方的抗辩方向：

1. **规模经济**：两家公司可以合并后端技术、客服、司机调度系统。声称节约 **$10\%$** 的固定成本 $\approx \$300M/年$。

2. **协同定价**（争议）：更精准的 ML 驱动定价 $\to$ 对消费者的最终价格可能下降。

3. **更强的议价力**（反向）：合并后对餐厅的议价更强，可以要求更低佣金 $\to$ 间接降低消费者价格。

**问题**：
- 规模经济 **能否通过** 内部增长实现？如果可以，合并不是必要条件
- 抗辩的 **定量证据** 需要专家证人建模
- **即使有效率，也必须能传递给消费者**——垄断者截留效率不算抗辩

### 六、进入壁垒 Entry Barriers

市场界定和单边效应之外的第三个核心分析：**新进入者能否约束合并后的涨价**？

配送行业的进入壁垒：
- **网络效应**：需要足够多的司机 + 餐厅 + 用户（鸡生蛋问题）
- **品牌认知**：用户习惯难改
- **高烧钱阶段**：早期需要大量补贴
- **监管复杂性**：劳动法（司机是员工还是独立承包商？）
- **数据优势**：现有平台有海量历史数据优化调度

**结论**：新进入 2–3 年内困难。

**反驳**：过去 5 年有 Grainhub、新兴本地平台进入 $\to$ 行业并非"封闭"。

### 七、DOJ 可能的 Remedy

如果 DOJ 不阻止合并，可能要求：

1. **资产剥离 (divestiture)**：在 HHI 最集中的 20 个城市剥离 ZipBite 或 FlashFood 业务
2. **行为承诺 (behavioral remedies)**：
   - 维持现有佣金率若干年
   - 允许司机多家平台工作
   - 数据共享给新进入者
3. **完全阻止**：最极端

## 建议

### 合并方的最优策略

1. **接受窄市场定义的可能性**——不要在这个战场硬刚，太难胜诉
2. **重点打"单边效应"**：论证 diversion ratio 被夸大，产品差异化（ZipBite 偏城市 + 午餐，FlashFood 偏郊区 + 晚餐）
3. **效率抗辩量化**：给出 **可验证、可传递** 的效率数字，最好由独立顾问（如 Brattle 或 Compass Lexecon）出具
4. **准备剥离方案**：主动提出在 10–15 个最敏感城市剥离 FlashFood 资产——这可能挽救 70% 的交易价值
5. **评估诉讼风险**：如果 DOJ 坚持完全阻止，合并方的期望值 vs 诉讼：
   - 诉讼胜率估计 $30\%$
   - 胜诉价值 $\$14B$
   - 败诉成本：法律费用 + 反向分手费 + 机会成本 $\approx \$500M$
   - 期望值 = $0.30 \times 14{,}000 - 0.70 \times 500 = 4{,}200 - 350 = \$3{,}850M$
   - 接受剥离方案的期望值（若剥离后 $70\%$ 保留）= $0.9 \times 0.70 \times 14{,}000 - 100 = \$8{,}720M$
   - **剥离方案更优**

### 时间线

- **Month 1–3**：完成内部经济分析，准备 submission
- **Month 3–6**：谈判期，提出剥离方案
- **Month 6–9**：如果 DOJ 不接受，决定诉讼 vs 放弃

## 核心教训

- **市场界定是反垄断案件的 $80\%$**——窄定义 $\to$ 高 HHI $\to$ 阻止；宽定义 $\to$ 低 HHI $\to$ 通过
- **SSNIP 是标准工具**，但 **critical loss** 和 **diversion ratio** 的估计需要 **可靠的消费者数据**——质量决定结果
- **单边效应** 是现代反垄断的核心：即使 HHI 不极端，如果合并双方是最近替代品，反竞争效应依然显著
- **效率抗辩必须可信且可传递给消费者**——否则在法官面前毫无说服力
- **剥离方案** 往往是最务实的出路——合并方宁可放弃 20% 的业务也要保住 80%
